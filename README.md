# liujing-first-project

## 项目介绍与使用方法
这是一个基于 Java 开发的拼图游戏项目，主要用于练习 GitHub 远程仓库的创建、代码推送以及 README 文档的规范化编写。

### 使用方法
1. **环境准备**：确保你的 MacBook Pro 已安装 JDK 17 和 IntelliJ IDEA。
2. **运行步骤**：
   1. 在 IDEA 中打开 `puzzlegame` 项目。
   2. 找到 `src` 目录下的主类并点击运行。

## 项目用途说明
本项目旨在通过实战练习掌握 Git 版本控制工具的基础指令，学习如何在团队协作中通过规范的 README 文档展示项目信息。

## 使用步骤（有序列表）
1. 登录 GitHub 账号并创建公开仓库。
2. 通过 IDEA 将本地项目代码推送到远程仓库。
3. 在网页端完善 README.md 说明文件。
